# 🚗 DriveX Showroom

A modern car showroom web application built using React.

This project allows users to explore premium cars, view detailed specifications, and book their preferred car through an interactive form.

---

## 📌 Project Overview

The main goal of this project is to build a responsive and user-friendly car showroom website using React.

Users can:

* Browse different cars
* View detailed information for each car
* Fill a booking form
* Contact for enquiries

---

## 🧩 Components Used

This project is structured using reusable React components:

* **Navbar** → Handles top navigation (Home, Cars, Booking, Contact)
* **Hero Section** → First landing section with heading, description, and CTA
* **CarList** → Displays all cars dynamically
* **CarCard** → Shows individual car details using props
* **Booking Form** → Handles user input and form submission using useState
* **Contact Section** → Displays personal contact details
* **Footer** → Displays closing section and branding
